> ViVeTool ID Converter

Convert between raw and obfuscated Windows Feature Configuration IDs used by ViVeTool / Albacore.ViVe.

Result

How it works

Obfuscate(id):
  x = id XOR 0x74161A4E
  x = SwapBytes(x)
  x = x XOR 0x8FB23D4F
  x = RotateLeft32(x, 1)
  x = x XOR 0x833EA8FF
  return x

Deobfuscate(id):
  x = id XOR 0x833EA8FF
  x = RotateRight32(x, 1)
  x = x XOR 0x8FB23D4F
  x = SwapBytes(x)
  x = x XOR 0x74161A4E
  return x

SwapBytes reverses byte order (big ↔ little endian).
