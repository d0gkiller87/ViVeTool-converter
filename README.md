# ViveTool ID Converter

Vibe-coded static tool to obfuscate/deobfuscate [ViVeTool](https://github.com/thebookisclosed/ViVe) IDs based on the official implementation: https://github.com/thebookisclosed/ViVe/blob/master/ViVe/ObfuscationHelpers.cs

Deobfuscated CFR (Controlled Feature Rollout) Feature IDs can be used in: `HKLM\SYSTEM\CurrentControlSet\Policies\Microsoft\FeatureManagement\Overrides`
